- Added a new dashboard page for quick actions and viewing recent games

- Added game bracket info

## v0.0.2 - February 24, 2026

- Added support for wincon cards
- Lots of UX tweaks.
- Lots of UX to map the table seating. Needs more mobile testing

## v0.0.1 - February 23, 2026

- Init

## What works

- Recording games from 2 to 6 player pods
- Importing legal commanders from Scryfall
- Remembers your commanders
- Basic stats: Win rate, win rate by turn
- Notes and basic wincons
- Saves your data locally, export a backup JSON, export a readable CSV

## What might not work

- Need to test partner thoroughly, especially if you swap partners around, Krark / Sakashima or Sakashima / Krark for example.

Feature requests and bugs, please file a PR or send me a note if you know me!
